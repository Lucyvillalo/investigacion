Diseñando con rejilla con Bootstrap

El sistema de cuadrícula de Bootstrap utiliza una serie de contenedores, filas y columnas para diseñar y alinear el contenido.
Está desarrollado con Flexbox y es totalmente responsivo.
Este ejemplo muestra de forma detallada cómo se construye la cuadrícula.
